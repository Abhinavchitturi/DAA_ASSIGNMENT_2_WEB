The Exact Method

Summary

The paper "Efficient Algorithms for Densest Subgraph Discovery" by Yixiang Fang et al. introduces an exact algorithm for identifying the densest subgraph in a graph with respect to edge-density or h-clique-density. The core innovation lies in transforming the densest subgraph discovery (DSD) problem into a maximum flow problem, combined with a binary search strategy to iteratively approximate the optimal density.

The algorithm leverages the concept of clique-degree and systematically constructs a flow network where vertices and (h–1)-clique instances are represented as nodes, and edge capacities encode density constraints. By performing repeated minimum s-t cut computations, the algorithm determines whether a subgraph exists exceeding a guessed density value. This process continues until convergence within a defined precision.

Time Complexity

The authors establish that the algorithm runs in:

$$ O\left( n \cdot \binom{d-1}{h-1} + \left(n|\Lambda| + \min(n, |\Lambda|)^3\right) \cdot \log n \right) $$

Where:

  • n = number of vertices
  • d = maximum degree
  • |Λ| = number of (h–1)-clique instances
  • First term: Time to enumerate all (h–1)-cliques
  • Second term: Cost per binary search iteration (flow network size and cut)
  • log n: Due to binary search iterations
